# ZK-ARCHE Privacy Considerations Skeleton

Status: draft skeleton.

Topics to cover:

- PID unlinkability assumptions;
- role authorization privacy;
- server lookup behavior and optional encrypted lookup hints;
- metadata leakage from transport, timing, packet size, and retry behavior;
- mTLS client-certificate linkability when TLS/mTLS binding is used;
- late enrollment and commissioner-grant privacy;
- anonymous-credential migration research;
- privacy regression tests for repeated authentication attempts.
