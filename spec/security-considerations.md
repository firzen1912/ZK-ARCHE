# ZK-ARCHE Security Considerations Skeleton

Status: draft skeleton.

Topics to cover before specification-grade claims:

- authentication goals and non-goals;
- prior-enrollment requirement for normal AUTH;
- transcript binding and downgrade prevention;
- unknown-key-share resistance;
- replay, reorder, stale-session, and cross-session behavior;
- anti-amplification and retry-cookie behavior;
- parser and malformed-input safety;
- RNG failure behavior;
- storage atomicity, restart, and rollback behavior;
- role-proof cryptographic review status;
- side-channel and constant-time requirements;
- formal-model coverage and explicit gaps.
